# 🎓 StudentHub

Modern web application designed to help students organize their study life efficiently — all in one place.

---

## ✨ Overview

**StudentHub** is a productivity platform that combines scheduling, note-taking, file storage, and AI-powered learning tools into a single intuitive interface.

It is built to simplify studying, improve organization, and help students focus on what really matters.

---

## 🚀 Features

### 📅 Rozvrh (Schedule)
- Weekly timetable (grid layout)
- Drag & drop subjects
- Color-coded subjects
- Teacher & subject details
- Odd / Even week switching
- Class detail view with notes and exams

---

### 📝 Zápisník (Notes)
- Organized by subjects and categories
- Clean, Notion-style editor
- Support for structured notes
- Optional drawing / sketch mode

---

### 📂 Materiály (Study Materials)
- File storage (PDFs, images, presentations)
- Folder-based organization (categories & subcategories)
- File search functionality
- Upload and preview system

---

### 🤖 AI Test Generator
- Generate tests from selected materials
- Multiple choice & open-ended questions
- Automatic evaluation
- Personalized learning recommendations

---

### 🔔 Smart Notifications
- Upcoming exams and deadlines
- Study reminders
- Integrated with schedule

---

### 🔐 Authentication
- Google OAuth login
- Simple and secure access
- User-specific data storage

---
